# `/404` — Trang không tìm thấy

**Status:** MVP
**Owner area:** System
**Source of truth:** `plan/requirement/page_function_matrix.md`, `plan/requirement/unified_database_schema.md`
**Build decision:** Build

## 1. Mục tiêu

Hiển thị trang thân thiện khi user truy cập URL không tồn tại, điều hướng user về trang phù hợp.

---

## 2. Trigger

| Trường hợp | Trigger |
|-----------|---------|
| URL không tồn tại | User nhập sai URL hoặc link cũ đã xóa |
| Course đã archived/deleted | User truy cập `/courses/[slug]` nhưng khóa đã ẩn |
| Lesson đã hidden | User truy cập `/learn/[course]/[lesson]` nhưng bài đã ẩn |
| Resource đã archived | User truy cập `/blog/[slug]` nhưng resource đã ẩn |
| Project đã hidden | User truy cập `/projects/[slug]` nhưng project đã ẩn |

## 5. CTA Actions

| CTA | Dẫn tới | Ghi chú |
|-----|---------|---------|
| `Về trang chủ` | `/` | CTA chính |
| `Xem khóa học` | `/courses` | CTA phụ |
| `Liên hệ hỗ trợ` | `/contact` | Nếu user cần giúp đỡ |
| Search box | `/blog?search=...` | Tìm kiếm resources |

---

## 6. Auto-redirect Logic

```text
Nếu URL match pattern cũ:
→ /courses/[slug] không tìm thấy → gợi ý khóa liên quan
→ /learn/[course]/[lesson] không tìm thấy → về /learn/[course]
→ /blog/[slug] không tìm thấy → về /blog
→ /projects/[slug] không tìm thấy → về /projects

Nếu không match pattern:
→ Hiển thị 404 chung
```
